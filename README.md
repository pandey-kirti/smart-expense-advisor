# Smart Expense Advisor

A web-based application to track, analyze, and predict personal expenses.

## Features
- Add and manage daily expenses
- Categorize spending (Food, Travel, etc.)
- Visualize expenses using charts
- Predict future expenses using a simple model
- User-friendly web interface

## Technologies Used
- Python
- Flask
- SQLite
- Machine Learning (basic)

## How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Initialize database:
   python init_db.py

3. Train model:
   python train_model.py

4. Run the app:
   python app.py

5. Open in browser:
   http://127.0.0.1:5000/
